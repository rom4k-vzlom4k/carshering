## Функциональность

### Пользователь:
- Вход по **номеру телефона**
- Просмотр списка **доступных машин**
- **Бронирование** машины
- **Завершение аренды** (в следующих версиях)
- Просмотр **текущей и прошлых аренд** (опционально)

### Администратор:
- Добавление новой машины
- Удаление машины
- Просмотр всех машин со статусом
